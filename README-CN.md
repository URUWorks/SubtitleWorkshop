# MPV 播放引擎

此软件使用了非常实用的mpv多媒体播放组件。
这意味着您需要安装"libmpv"组件：

- Windows:复制"mpv-2.dll"到该软件主程序文件夹下。
- macOS: 从此仓库中的"./Others/mpv-macos-dylibs.zip"下载预编译好的文件，然后解压文件到"/usr/local/lib"。
- Linux: 根据发行版安装"mpv"。
